UX/UI

Kapas

Editando fotografías con un par de taps de tu dedo.

En 2015 trabajamos junto con el equipo de Blu27 Kapas, una aplicación que permite editar fotografías usando diferentes librerías de elementos como iconos, textos, emojis, filtros, etc. permite enriquecer las imágenes usando ese tipo de extras. Trabajamos la idea centrados en el nombre de la App, usando “Capas” sobre cada fotografía que editábamos, para culminar con la capacidad de compartir cada imagen en las diferentes redes sociales.

UX

La experiencia de usuario es siempre nuestro foco principal, logramos una experiencia fluida usando tarjetas para las librerías y generando interacciones naturales mediante swipes de elementos, para facilitar los cambios entre secciones.

UI

Amamos los colores brillantes y esta aplicación nos permitía jugar con ellos, buscamos darle relevancia a los aspectos esenciales de la interfaz, de igual modo pudimos proponer un esquema de colores novedoso ya que en ese momento la Aplicación no contaba con un branding establecido por el cliente.

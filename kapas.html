UX/UI

Kapas

Editing photos in a couple of taps

In 2015 we worked with Blu27 team: Kapas App. An App that lets you edit photos using different element libraries like texts, emojis, filters, icons, etc. It allows you to enrich the images using these extra features. We worked the idea based on the App name Kapas (“Capas" in spanish which means “layers”) over every photograph that we edited. We made the app have the capacity to share every image in the different social media networks.

UX

User experience is always our main focus, we obtain a fluid experience using cards for the libraries and generating natural interaction through element swipes, to facilitate changes between sections.
